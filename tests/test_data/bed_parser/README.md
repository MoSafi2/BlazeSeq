# BED test data (Biopython BLAT)

Test files vendored from the [Biopython Tests/Blat](https://github.com/biopython/biopython/tree/master/Tests/Blat) directory, used for correctness testing of BlazeSeq's BED parser.

## Included files

- `bed3.bed` … `bed12.bed`: Minimal BED fixtures with 3 through 12 fields.

## Notes

- These fixtures are **tab-delimited** (as required by the BED specification and BlazeSeq's parser).
- Biopython also includes `*.bb` BigBed binaries in `Tests/Blat`; we do not vendor those here.
